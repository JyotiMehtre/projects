# How LangGraph Works – Questions & Answers

### 1. What is a Node?
A Node is simply a Python function that performs one step in the workflow.

### 2. What is State?
State is the data passed from one node to another.

### 3. What are Edges?
Edges define the order of execution between nodes.

### 4. How does the workflow run?
The graph starts at START → executes nodes in order → ends at END.

### 5. Why use LangGraph?
LangGraph helps build multi-step AI workflows that are modular, traceable, and maintainable.
