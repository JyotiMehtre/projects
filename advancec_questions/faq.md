### 1. What is the difference between LangChain and LangGraph?
LangChain is for chains and tools.  
LangGraph is for workflows and stateful agents.

### 2. Can we add loops?
Yes, conditional edges allow loops.

### 3. Can we use local models?
Yes, using Ollama or Transformers.

### 4. What is multi-agent workflow?
Multiple agents (nodes) doing different tasks in the same graph.
